## Description

[Nest](https://github.com/nestjs/nest) framework TypeScript starter repository.

## Installation

```bash
$ pnpm install
```

## Running the app

```bash
# development
$ pnpm run start

# watch mode
$ pnpm run start:dev

# production mode
$ pnpm run start:prod
```

## Test

```bash
# unit tests
$ pnpm run test

# e2e tests
$ pnpm run test:e2e

# test coverage
$ pnpm run test:cov
```

 - tsconfig.json 和 tsconfig.build.json —— typescript 配置文件，typescript 编译就是根据这个文件，一般不作修改 - package.json —— nodejs 项目必备文件，管理项目依赖、启动命令等等重要配置
